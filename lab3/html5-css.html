HTML5 e CSS

HTML5 (Hypertext Markup Language, atualmente na sua 5° versão) é uma linguagem de marcação para a World Wide Web e é uma tecnologia chave da Internet, originalmente proposto pelo Opera Software. É a quinta versão da linguagem HTML. Esta nova versão traz consigo importantes mudanças quanto ao papel do HTML no mundo da Web, através de novas funcionalidades como semântica e acessibilidade.

Header ✔ Indica cabeçalho(normalmente com logótipos). Usado para os títulos das diversas páginas.
Nav ✔ Indica menu de navegação com links para as páginas(só deve haver um por página). Usado para a criação do menu de maneira a navegar entre páginas.
Main ❌ Indica conteúdo principal e único, que não se repete.
Article ❌ Indica porção do documento que pode ser vista de forma independente.
Section ✔ Especifica uma parte do conteúdo, usada em conjunto com etiquetas h1|h2|… Usado para a criação de secções nas páginas.
Asside ❌ Indica anotações, publicidade ou outros assuntos desconexos.
Footer ✔ Indica rodapé da secção (com copyright). Usado para indicar o autor do website.

CSS

Cascading Style Sheets (CSS) é um mecanismo para adicionar estilo (cores, fontes, espaçamento, etc.) a um documento web.

Universal ✔ O seletor universal (*) aplica o estilo a todos os elementos da página. Utilizado para trocar a fonte do site.
Tipo ✔ O seletor de tipo usa o nome do marcador a estilizar. Utilizado para utilizar diferentes cores nos títulos.
Classe ✔ O seletor de classe associa um nome a um estilo. Utilizado para tranformar as legendas bold.
id ✔ O atributo id permite identificar univocamente um elemento do HTML(muito útil no JavaScript). Utilizado para colocar uma cor no background do título das páginas.
Seletores de Atributo ✔ Seletores que se aplicam a atributos de elementos.
Seletores de Pseudo-classe ✔ Seletor associado ao estado de um elemento. Usado para estilizar os links presentes na página.
Seletores de Pseudo-classe ✔ Usado para aumentar a primeira letra de cada parágrafo.


Seletores descendentes -> div p ❌ Todos os p que estão “dentro” de um div
Seletores filhos -> div > p ❌ Todos os p que estão “logo dentro” de div
Seletores adjacentes -> div ~ p ❌ Todos os p que estão colocados depois de um div
Seletores imediatamente adjacentes -> div + p ❌ Todos os p que estão colocados “imediatamente” depois de um div
Agrupamento de seletores, separando-os por vírgulas -> div,p{ color: red; } ❌ Os div e p ficacarão com fonte vermelha
